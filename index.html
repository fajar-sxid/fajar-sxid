apa itu deface?

Defacing adalah sebuah kegiatan yang dilakukan untuk mengubah tampilan pada suatu website dengan cara menyisipkan file pada server website tersebut. Teknik ini bisa dilakukan karena adanya celah keamanan yang ada pada sebuah website.

apa itu Front End?

Front-End adalah sebuah bagian dari website yang menyuguhkan tampilan kepada user. Bagian ini dibangun menggunakan html (HyperText Markup Language), CSS (Cascading Style Sheets), dan JavaScript sehingga sebuah URL bisa berfungsi dan menampilkan website yang baik

apa itu Back-End?

Back-End adalah orang Yang mampu menguasai bahasa pemrograman php,mysql,python dll
